# Cosmic Explorer
## Explore the daily images/videos from any time form NASA!

This project from the tutorial was fetching some data. I wanted to add a spin, a touch of FLAVOR if you will, so I decided to allow the user to pick ANY date and retireve the image and description of it from that day. And, if there's a video, you can play that as well. Furthermore, you can pick any date, without anything breaking!
